Installer GeoWeb Kaffrine

Version PWA: installation rapide sur mobile et ordinateur.

Android / Chrome

  1. Ouvrir l'application.
  2. Menu navigateur.
  3. Choisir "Ajouter à l'écran d'accueil".

iOS / Safari

  1. Ouvrir l'application dans Safari.
  2. Bouton Partager.
  3. Choisir "Sur l'écran d'accueil".
